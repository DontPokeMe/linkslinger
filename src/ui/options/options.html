About LinkSlinger

LinkSlinger is a powerful browser extension designed to streamline link management and enhance your browsing efficiency. Part of the dontpoke.me OSINT toolkit, LinkSlinger helps you quickly select, copy, and open multiple links with simple mouse gestures.

Open Source & Privacy-Focused

LinkSlinger is built by security researchers with privacy and transparency at its core. As an open-source project, our code is publicly available for review, ensuring you can trust what's running in your browser.

Key Features

  • Bulk Link Selection: Select multiple links at once using customizable mouse gestures.
  • Flexible Actions: Open links in new tabs, copy URLs to clipboard, or bookmark in bulk.
  • Context Menu Integration: Right-click options for quick link management.
  • RegEx Filtering: Advanced link filtering with regular expressions.
  • Privacy First: No tracking, no data collection, no external calls.

Part of the OSINT Toolkit

LinkSlinger is one of several privacy-focused tools in the dontpoke.me suite, designed for researchers, analysts, and power users who need efficient, secure browsing tools.
